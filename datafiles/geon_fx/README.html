Geon FX Asset Help

What is Geon FX?

Geon FX is the newest and the most advanced particle editor for GameMaker: Studio and Game Maker Studio 2.

It allows you to create complex particle effects on Windows or Mac and play them in your project easily.

Geon FX comes with plenty of effects for you to test and examine.

Where is Geon FX?

There are two archives in Included Files - editor folder. One is for macOS, another for Windows.

Extract archives and run .exe (on Windows) or .app (on macOS).

Press F1 on keyboard or click ? icon at the top-right corner of the window to show editor Help.

How to play Geon FX effects?

You can either export your effect to GML in Geon FX editor then copy and paste resulting code into your project. Or use our custom loading scripts.

We have designed a very simple system for you to use Geon FX effects. That’s how it works:

  1. Import Geon_FX_Particle_Editor extension to your project;
  2. Import all sprites from geon_fx folder;
  3. Add .geon effects as Included Files;
  4. Load them using geon_effect_add(filename, -1) function (check our examples);
  5. Play any effect once with geon_effect_burst(effect_ind, x, y) or continuously with geon_effect_stream(effect_ind, x, y, enable).

Examples

There are several rooms in the project. You can change their order and run whichever you want first.

  1. rm_geon_test_stream contains obj_effect_stream which streams several Geon FX effects. You can cycle through them with Left Click.
  2. rm_geon_test_burst and obj_effect_burst show how to load and play Geon FX effect once. Left Click to create an explosion.
  3. rm_geon_test_concurrent_effects shows how to play a dozen Geon FX effects simultaneously. Left Click to create explosions. 1-8 to switch fire streams on/off.
  4. obj_effect_gml_export_rain and obj_effect_gml_export_fire are two objects in corresponding rooms. They are illustrating GML export function from Geon FX editor.

Functions List

  • geon_system_init(ps) — Geon FX automatically creates the Particle System and holds it's id in global.geonfx_ps variable. You can destroy the default particle system and change it with your own by assigning it to global.geonfx_ps
  • geon_system_destroy() — call it if you want to clear the Geon FX effect system by removing all effects from the memory.
  • geon_effect_add(filename, ps, [sprite_array], [ps_add]) — Loads new effect from a file fname and uses particle system ps to play it. [sprite_array] argument is there to replace the effect sprites with custom sprites from your project. [ps_add] allows you to set an alternative particle system for all effects that use additional blending.
  • geon_effect_burst(effect_ind, x, y, [direction|gravity_direction|speed_multiplier|gravity_multiplier]) - Play effect with index effect_ind at x, y once. Last argument allows to set direction and speed of the entire effect in realtime.
  • geon_effect_stream(effect_ind, x, y, enable, [direction|gravity_direction|speed_multiplier|gravity_multiplier]) - Enable or disable effect streaming for effect_ind at x, y coordinates. Also allows to rewrite default direction and speed settings in realtime.
  • geon_effect_is_streamed(effect_ind) - Checks whether effect with index effect_ind is being streamed
  • geon_effect_delete(effect_ind, clear_pt) - Removes effect with index effect_ind from memory. If ( clear_pt == true ) particle types will be destroyed too.
